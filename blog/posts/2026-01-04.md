# Modern JavaScript Features

JavaScript has evolved significantly over the years. Let's explore modern ES6+ features that make code cleaner and more powerful.

## Destructuring

Extract values from arrays and objects elegantly.

```javascript
// Array destructuring
const [first, second, ...rest] = [1, 2, 3, 4, 5];
console.log(first); // 1
console.log(rest);  // [3, 4, 5]

// Object destructuring
const user = { name: 'John', age: 30, city: 'NYC' };
const { name, age } = user;

// With renaming
const { name: userName, age: userAge } = user;

// With default values
const { country = 'USA' } = user;
```

## Spread Operator

Expand iterables and objects.

```javascript
// Array spreading
const arr1 = [1, 2, 3];
const arr2 = [...arr1, 4, 5]; // [1, 2, 3, 4, 5]

// Object spreading
const obj1 = { a: 1, b: 2 };
const obj2 = { ...obj1, c: 3 }; // { a: 1, b: 2, c: 3 }

// Function arguments
function sum(...numbers) {
  return numbers.reduce((a, b) => a + b, 0);
}
sum(1, 2, 3, 4); // 10
```

## Arrow Functions

Concise function syntax with lexical `this`.

```javascript
// Traditional
function add(a, b) {
  return a + b;
}

// Arrow function
const add = (a, b) => a + b;

// With single parameter
const double = x => x * 2;

// With multiple lines
const complex = (x, y) => {
  const result = x * y;
  return result + 10;
};
```

## Template Literals

String interpolation and multi-line strings.

```javascript
const name = 'Alice';
const age = 25;

// String interpolation
const greeting = `Hello, ${name}! You are ${age} years old.`;

// Multi-line strings
const message = `
  This is a multi-line
  string that preserves
  formatting.
`;

// Tagged templates
function highlight(strings, ...values) {
  return strings.reduce((result, str, i) => {
    return `${result}${str}<mark>${values[i] || ''}</mark>`;
  }, '');
}

const text = highlight`Hello ${name}, welcome!`;
```

## Async/Await

Clean asynchronous code.

```javascript
// Promise-based
fetch('/api/users')
  .then(response => response.json())
  .then(users => console.log(users))
  .catch(error => console.error(error));

// Async/await
async function getUsers() {
  try {
    const response = await fetch('/api/users');
    const users = await response.json();
    console.log(users);
  } catch (error) {
    console.error(error);
  }
}

// Parallel requests
async function getAllData() {
  const [users, posts, comments] = await Promise.all([
    fetch('/api/users').then(r => r.json()),
    fetch('/api/posts').then(r => r.json()),
    fetch('/api/comments').then(r => r.json())
  ]);
  return { users, posts, comments };
}
```

## Optional Chaining

Safely access nested properties.

```javascript
const user = {
  name: 'John',
  address: {
    city: 'NYC'
  }
};

// Without optional chaining
const zipCode = user && user.address && user.address.zipCode;

// With optional chaining
const zipCode = user?.address?.zipCode;

// For function calls
const result = obj.method?.();

// For arrays
const firstItem = arr?.[0];
```

## Nullish Coalescing

Provide defaults for null/undefined.

```javascript
// Logical OR (falsy values)
const value1 = 0 || 'default'; // 'default'
const value2 = '' || 'default'; // 'default'

// Nullish coalescing (only null/undefined)
const value3 = 0 ?? 'default'; // 0
const value4 = '' ?? 'default'; // ''
const value5 = null ?? 'default'; // 'default'
```

## Array Methods

Powerful array manipulation.

```javascript
const numbers = [1, 2, 3, 4, 5];

// map - transform elements
const doubled = numbers.map(n => n * 2);

// filter - select elements
const evens = numbers.filter(n => n % 2 === 0);

// reduce - aggregate
const sum = numbers.reduce((acc, n) => acc + n, 0);

// find - first match
const found = numbers.find(n => n > 3);

// some - check if any match
const hasEven = numbers.some(n => n % 2 === 0);

// every - check if all match
const allPositive = numbers.every(n => n > 0);
```

## Modules

Organize code into reusable modules.

```javascript
// math.js
export const add = (a, b) => a + b;
export const multiply = (a, b) => a * b;
export default class Calculator {
  // ...
}

// main.js
import Calculator, { add, multiply } from './math.js';
// or
import * as math from './math.js';
```

These features make JavaScript more expressive and maintainable. Start using them today!

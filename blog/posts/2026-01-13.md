# TypeScript Tips and Tricks

TypeScript has become the standard for building robust JavaScript applications. Here are some tips to level up your TypeScript game.

## 1. Use Type Inference

Let TypeScript infer types when possible:

```typescript
// Good
const name = "John"; // string is inferred
const age = 30; // number is inferred

// Unnecessary
const name: string = "John";
```

## 2. Leverage Union Types

Union types provide flexibility:

```typescript
type Status = "pending" | "success" | "error";

function handleStatus(status: Status) {
  // TypeScript ensures only valid values
}
```

## 3. Use Utility Types

TypeScript provides powerful utility types:

- `Partial<T>`: Make all properties optional
- `Required<T>`: Make all properties required
- `Pick<T, K>`: Select specific properties
- `Omit<T, K>`: Exclude specific properties

```typescript
interface User {
  id: number;
  name: string;
  email: string;
}

type UserPreview = Pick<User, "id" | "name">;
```

## 4. Strict Mode is Your Friend

Always enable strict mode in `tsconfig.json`:

```json
{
  "compilerOptions": {
    "strict": true
  }
}
```

## Conclusion

These tips will help you write more maintainable and type-safe code. Happy coding!

# Building Responsive Designs

Creating websites that work beautifully on all devices is essential. Let's explore modern responsive design techniques.

## Mobile-First Approach

Start with mobile styles and progressively enhance for larger screens.

```css
/* Base styles (mobile) */
.container {
  padding: 1rem;
  width: 100%;
}

/* Tablet */
@media (min-width: 768px) {
  .container {
    padding: 2rem;
    max-width: 720px;
    margin: 0 auto;
  }
}

/* Desktop */
@media (min-width: 1024px) {
  .container {
    max-width: 960px;
  }
}

/* Large desktop */
@media (min-width: 1280px) {
  .container {
    max-width: 1200px;
  }
}
```

## Fluid Typography

Scale text smoothly across screen sizes.

```css
/* Using clamp() */
h1 {
  font-size: clamp(2rem, 5vw, 4rem);
}

p {
  font-size: clamp(1rem, 2vw, 1.125rem);
}

/* Alternative with calc() */
h2 {
  font-size: calc(1.5rem + 1vw);
}
```

## Responsive Images

Serve optimal images for each device.

```html
<!-- Using srcset -->
<img 
  src="image-800.jpg"
  srcset="
    image-400.jpg 400w,
    image-800.jpg 800w,
    image-1200.jpg 1200w
  "
  sizes="(max-width: 600px) 100vw, 50vw"
  alt="Description"
>

<!-- Using picture -->
<picture>
  <source 
    media="(min-width: 1024px)" 
    srcset="large.jpg"
  >
  <source 
    media="(min-width: 768px)" 
    srcset="medium.jpg"
  >
  <img src="small.jpg" alt="Description">
</picture>
```

## Container Queries

Style components based on container size, not viewport.

```css
.card-container {
  container-type: inline-size;
  container-name: card;
}

.card {
  display: flex;
  flex-direction: column;
}

@container card (min-width: 400px) {
  .card {
    flex-direction: row;
  }
  
  .card-image {
    width: 40%;
  }
}
```

## Flexible Grid Layouts

Create responsive grids without media queries.

```css
.grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 2rem;
}

/* Cards automatically wrap based on available space */
```

## Responsive Navigation

Transform navigation for small screens.

```css
/* Mobile menu */
.nav {
  display: none;
  flex-direction: column;
  position: absolute;
  top: 60px;
  left: 0;
  right: 0;
  background: white;
  box-shadow: 0 2px 8px rgba(0,0,0,0.1);
}

.nav.is-open {
  display: flex;
}

/* Desktop menu */
@media (min-width: 768px) {
  .nav {
    display: flex;
    flex-direction: row;
    position: static;
    box-shadow: none;
  }
  
  .menu-toggle {
    display: none;
  }
}
```

## Aspect Ratio

Maintain consistent proportions.

```css
.video-wrapper {
  aspect-ratio: 16 / 9;
  width: 100%;
}

.square {
  aspect-ratio: 1 / 1;
}
```

## Viewport Units

Utilize viewport-relative units.

```css
/* Full viewport height */
.hero {
  height: 100vh;
  min-height: 500px; /* Fallback */
}

/* Dynamic viewport units (mobile safe) */
.section {
  height: 100dvh; /* Dynamic viewport height */
}

/* Width relative to viewport */
.wide-text {
  font-size: 5vw;
  max-font-size: 3rem;
}
```

## Responsive Spacing

Scale spacing with viewport size.

```css
.section {
  /* Small screens */
  padding: 2rem 1rem;
}

@media (min-width: 768px) {
  .section {
    /* Medium screens */
    padding: 4rem 2rem;
  }
}

@media (min-width: 1024px) {
  .section {
    /* Large screens */
    padding: 6rem 3rem;
  }
}

/* Or use clamp */
.fluid-section {
  padding: clamp(2rem, 5vw, 6rem) clamp(1rem, 3vw, 3rem);
}
```

## Testing Responsive Designs

- Test on real devices
- Use browser DevTools
- Check portrait and landscape
- Test touch interactions
- Verify text readability
- Check performance on mobile networks

## Common Breakpoints

```css
/* Small phones */
@media (max-width: 480px) { }

/* Phones */
@media (min-width: 481px) { }

/* Tablets */
@media (min-width: 768px) { }

/* Laptops */
@media (min-width: 1024px) { }

/* Desktops */
@media (min-width: 1280px) { }

/* Large desktops */
@media (min-width: 1536px) { }
```
